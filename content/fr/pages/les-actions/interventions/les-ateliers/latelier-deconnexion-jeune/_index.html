L'atelier déconnexion est ludique, bienveillant et participatif. Le contenu pédagogique et le vocabulaire sont adaptés en fonction de l'âge des participants.

Pour se recentrer et expérimenter nos facultés d'attention.

alt: >- credit: >- - title: >- Quizz, sondage mouvant description: >-

Par le mouvement, nous questionnons les pratiques numériques des participants : quels types d'écrans ? fréquence d'usage ? temps passé ?

alt: >- credit: >- - title: >- Jeu de société Deconnexio description: >-

Pour aborder de façon ludique les effets de la surexposition aux écrans.

alt: >- credit: >- - title: >- Débat description: >-

De manière participative, nous amenons les participants à réfléchir sur les effets des écrans

alt: >- credit: >- - title: >- Captologie description: >-

Découverte du concept et définition des enjeux de l'économie de l'attention

alt: >- credit: >- - title: >- Procès fictif des applications description: >-

Jeu de rôles pour élaborer des arguments et pistes de solutions

alt: >- credit: >- - title: >- Défi description: >-

Un défi à relever à la maison !
