DR - Sarah Limorte anime un atelier à l'école Malpassé les Oliviers à Marseille, mars 2022.

meta_description: >- summary: >-

Ciblé sur les enfants de 5 à 7 ans, cet atelier est axé sur deux aspects essentiels de la vie de l'enfant : le sommeil et la gestion des émotions. Il est conçu pour durer une heure.

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 47 text: >- 1 minute contents: - kind: heading title: >- Contenus slug: "contenus" position: 1 rank: 2 - kind: block template: features title: >- ranks: self: false children: 3 data: description: >- elements: - title: >- Quizz description: >-

A partir de tableaux illustrés, nous questionnons les enfants sur leurs pratiques numériques : quels types d'écrans ? à quel moment de la journée ? où ? avec qui ?

alt: >- credit: >- - title: >- Jeux sur l'attention description: >-

Pour prendre conscience de son corps et de ses émotions

alt: >- credit: >- - title: >- Puzzle et cartes images description: >-

Utiliser le jeu et les images pour aborder l'importance et les bienfaits du sommeil

alt: >- credit: >- - title: >- Lecture de conte description: >-

Faire vivre aux enfants l'expérience de l'écoute et mobiliser leur imagination autour de la thématique

alt: >- credit: >- - title: >- Boîte à idées description: >-

Donner des inspirations d'activités déconnectées

alt: >- credit: >- - kind: heading title: >- Durée slug: "duree" position: 2 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

L'atelier peut durer entre une et quatre heures, sous la forme d'une ou deux séances.
